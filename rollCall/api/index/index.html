RollCall

Attendance for your Telegram group, made simple. Open a rollcall, let everyone vote in, out or maybe, and track who shows up.

➕ Add to a Telegram group

What you get

🗳️

One-tap voting

Members reply /in, /out or /maybe — or tap the panel buttons.

📊

Stats & streaks

Per-member attendance rates, streaks and personal bests, tracked automatically.

🌐

Vote from the web

A permanent web link lets people vote even when Telegram is down.

⏰

Reminders & templates

Auto-close rollcalls, send reminders, and reuse recurring event templates.

Get started in 30 seconds

  1. Add @{{BOT_USERNAME}} to your group.
  2. An admin runs /src to start a rollcall.
  3. Everyone votes with /in, /out or /maybe.
  4. Run /erc to close it and record the stats.
Run /help in your group for the full command list.
