⚙️ Admin dashboard — help

Manage your groups, rollcalls, templates and settings from one place. Sign in with an admin API token issued from Telegram.

Tokens

A global token (chat_id = 0) sees and edits every group. A group-specific token can only manage the group it was issued for. Generate tokens from the bot — run /help admin in your group.

Groups sidebar

Pick a group on the left to manage it. On mobile, tap ☰ to open the menu. Use ↻ to refresh the list.

Rollcalls

Start a new rollcall, view live in/out/maybe lists, or end one — even when Telegram is unreachable. Ending records attendance and stats; cancelling discards it without affecting streaks.

Templates & schedules

Save recurring events as templates and let them auto-open on a weekly schedule, so you don't re-create the same rollcall each week.

Settings

Per-group timezone, attendance limit, ghost tracking and more. Changes apply immediately.
